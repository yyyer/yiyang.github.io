使用IDM + 油猴插件 快速下载百度网盘文件

发表于 2019-04-20 | 分类于 教程

序

百度网盘客户端下载的限速让人十分蛋疼，快速下载百度网盘的文件呢？

1.概述

IDM支持从网站上抓取文件来下载，油猴插件对网页有着许多强大和实用的功能。

下载的原理是使用油猴插件来绕过网页版百度网盘下载必须使用客户端的限制，转到浏览器下载，然后再使用IDM抓取下载，提升下载速度。

2.IDM

IDM (Internet Download Manager)是综合数据复用器是一种数据复用设备，它可以将多路RS232、RS485及数字语音等多种数据复用到E1传输通道或光传输通道内，实现不同类型数据在同一通道内的复用、传输。

总之，意思就是IDM下载速度很快，可以满宽下载。

2.1 IDM的下载

因为IDM是要付费注册的，所以我提供了IDM破解版(在文末)

有钞能力的可以上官网付费下载。

注：接下来讲解是针对破解版的安装，从官网上下载的安装方法就是傻瓜式安装。

2.1安装IDM

打开我提供的IDM破解版.exe

傻瓜式安装。

2.2IDM的使用

IDM使用起来也非常简单，当你打开浏览器，看到插件中有IDM，就说明IDM在浏览器中安装成功。

然后你就不要管它了。

3.油猴插件

Tampermonkey可替网页加入些新功能、修正网页错误、组合来自不同网页的数据、或者数繁不及备载的其他功能。写的好的Tampermonkey脚本甚至可让其输出与被修改的页面集成得天衣无缝，像是原本网页里的一部分。

3.1油猴插件的安装

打开谷歌浏览器(注意，一定要用谷歌浏览器！！！)，进入插件页面

在我提供的文件包(在文末)里找到Tampermonkey_v4.5.0.crx，把它拖入浏览器即可。

3.2在油猴插件中安装脚本

打开Greasy Fork

点击安装此脚本 → 安装

安装完成后在已安装脚本查看脚本是否启动

4.使用

在网页版百度网盘上打开你要下载的文件

点击右上角的下载助手

点击里面的直接下载

这时IDM自动跳出显示开始下载，点击开始下载即可

5.常见问题与处理

Q:为什么我的浏览器不能安装IDM？

A:通常情况下任何浏览器都可以安装IDM，但是最好使用Chrome，没有为什么。

Q:怎么知道IDM已经安装？

A:检查IDM是否已经安装可以在页面的任何地方右键，看到以下内容就代表安装成功。

Q:为什么有时候下载会在浏览器上直接下载而不是用IDM？

A:很少见，但是我觉得是因为下载文件太小的原因。

Q:为什么IDM有时会跳出一个全是英文的框框？

A:因为是破解版，所以会自动弹出，处理方法就是点取消或者直接关闭。千万不要点确定，有时点了电脑就会重新启动！！！
